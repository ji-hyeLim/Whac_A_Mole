## :joystick: 게임소개
<br>
화면에 보이는 start 버튼을 누른 후 두더지가 나타나면 클릭으로 잡으면 점수가 올라갑니다.

제한 시간이 끝나면 경고창이 뜨며 게임이 마무리됩니다.
